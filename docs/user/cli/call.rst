call
====

The ``call`` namespace provides you an interface to call common MarkLogic resources.

.. code-block:: none

    Available commands for the "call" namespace:
      call eval  Sends a GET request to the /v1/eval endpoint
      call logs  Sends a GET request to the /manage/v2/logs endpoint
